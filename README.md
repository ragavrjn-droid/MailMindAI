# MailMindAI

An autonomous AI agent that monitors Gmail Inbox and Spam folders, classifies interview-related emails using a local LLM, extracts structured information, and sends WhatsApp notifications.

## Features

- Gmail Inbox Monitoring
- Spam Monitoring
- Local AI (Ollama)
- Interview Extraction
- WhatsApp Notifications
- SQLite Tracking

## Tech Stack

- Python
- Gmail API
- Ollama
- Qwen
- SQLite
- Playwright
