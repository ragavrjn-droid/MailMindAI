# MailMindAI 🤖

An autonomous AI agent that monitors Gmail Inbox and Spam folders, understands interview-related emails using a local Large Language Model (LLM), extracts structured information, and sends real-time WhatsApp notifications.

---

## Features

- 📧 Gmail Inbox Monitoring
- 🗑 Spam Folder Monitoring
- 🧠 Local AI using Ollama + Qwen
- 📅 Interview Information Extraction
- 💬 WhatsApp Notifications
- 🗄 SQLite State Management
- 🔒 AI Guardrails
- ⏰ Continuous Monitoring

---

## Tech Stack

- Python
- Gmail API
- Ollama
- Qwen 2.5
- SQLite
- Playwright
- APScheduler
